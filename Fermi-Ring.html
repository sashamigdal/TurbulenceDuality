The Markov Chain and its Fermionic Representation

Here is a new representation of the Euler ensemble, leading us to the exact analytic solution below.

We start by replacing independent random variables \( \sigma \) with fixed sum by a Markov process, as suggested in [Migdal, 2023]. We start with \( n \) random values of \( \sigma_i = 1 \) and remaining \( N-n \) values of \( \sigma_i = -1 \). Instead of averaging over all of these values simultaneously, we follow a Markov process of picking \( \sigma_N, \dots, \sigma_1 \) one after another.

At each step, there will be \( M = N,\dots, 0 \) remaining \( \sigma \). We get a transition \( n \Rightarrow n-1 \) with probability \( \frac{n}{M} \) and \( n \Rightarrow n \) with complementary probability.

Multiplying these probabilities and summing all histories of the Markov process is equivalent to computing the product of the Markov matrices:

\[ \prod_{M=1}^{N} Q(M); \] \[ Q(M) \lvert n\rangle = \frac{M-n}{M} \lvert n\rangle + \frac{n}{M}\lvert n-1\rangle; \]

The expectation value of some function \( \hat{X}(\left\{\sigma\right\}) \) reduces to the matrix product:

\[ \mathbb{P}[\hat{X}] = \sum_{n=0}^{N_+} \left\langle n \right\rvert \left(\prod_{M=1}^{N} \hat{Q}(M)\right)\cdot \hat{X} \cdot \left \lvert N_+\right\rangle; \] \[ \hat{Q}(M)\cdot \hat{X}\lvert n\rangle = \frac{n}{M} \hat{X}(\sigma_{M}\to 1)\lvert n-1\rangle + \frac{M-n}{M} \hat{X}(\sigma_{M}\to -1)\lvert n\rangle. \]

Now, we introduce the Fermionic representation, where we define the following commutation relations:

\[ \left[a_i, a^\dagger_j\right]_+ = \delta_{i j}; \] \[ \left[a_i, a_j\right]_+ =\left[a^\dagger_i, a^\dagger_j\right]_+ =0. \]

The number \( n(M) \) of positive sigmas \( \sum_{l=1}^M\delta[\sigma_l -1] \) coincides with the occupation number of these Fermi particles:

\[ \hat n(M) = \sum_{l=1}^M \hat{\nu}_l. \]

This leads to the representation:

\[ \hat{Q}(M) = \hat{\nu}_M \frac{\hat n(M)}{M} + (1 - \hat{\nu}_M)\frac{M-\hat n(M)}{M}. \]

Next, we define the Wilson loop:

\[ \Psi(\gamma, C) = \frac{\langle \hat{W}[C] \rangle_{\mathcal{E}(N)}}{\langle \hat{W}[0] \rangle_{\mathcal{E}(N)}}; \] \[ \hat{W}[C] = \text{tr} \left(\hat Z(q r)\exp{\frac{i \gamma \hat \Gamma}{\nu} } \prod_{M=1}^{N} \hat{Q}(M)\right). \]

Using the lemma:

Path Integral Over Markov Histories

The product \( \Pi_N \) of the transition probabilities of a given Markov process history can be written as:

\[ \Pi_N = \exp{ N \Lambda_N}; \] \[ \Lambda_N = \frac{1}{N}\sum_l G_l; \]

where:

\[ G_l = \Delta n_+ \log\left(\frac{n_+}{n_+ + n_-}\right) + \Delta n_- \log\left(\frac{n_-}{n_+ + n_-}\right). \]

\[ n_\pm = N f_\pm(\xi), \quad f_\pm(\xi) = \sum_{k =1}^{\lfloor N \xi \rfloor} \frac{\nu_k}{N}. \]
